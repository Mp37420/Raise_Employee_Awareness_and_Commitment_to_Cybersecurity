# Raise Employee Awareness and Commitment to Cybersecurity

## 📌 Overview
This project focuses on improving employee cybersecurity awareness and promoting secure behavior within the organization.

## 🎯 Objectives
- Increase awareness of cybersecurity risks
- Promote secure practices among employees
- Strengthen compliance with security policies

## 📂 Project Structure
- `/docs` → User Charter (roles & responsibilities)
- `/training` → Security Awareness Training (presentation)
- `/reference` → Quick Reference Guide (Open Pharma)

## 🛠 Key Activities
- Developed user charter aligned with organizational security policies
- Designed employee awareness training materials
- Created quick reference guides for secure practices

## 📊 Outcome
- Improved employee awareness of cybersecurity risks
- Reduced human-related vulnerabilities
- Strengthened compliance posture

## 🔐 Framework Alignment
- ISO 27001 (Security Awareness Controls)
- NIST Cybersecurity Framework (Awareness & Training)
